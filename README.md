# 🚀 HireSphere – Smart Job Portal with Skill-Based Matching

## 📌 Project Overview

HireSphere is a modern web-based job portal designed to connect job seekers and recruiters efficiently. It provides a smart and user-friendly platform where users can search for jobs based on skills, location, and job type.

The system includes an AI-based resume matcher that analyzes resumes and suggests relevant job opportunities, reducing manual effort and improving accuracy.

---

## 🎯 Objectives

* Simplify the job search process
* Provide smart job recommendations
* Enable easy job filtering (skills, location, type)
* Help recruiters find suitable candidates
* Improve overall user experience

---

## 🌐 Features

* 🔍 Job search with filters
* 📄 Resume upload and analysis (AI-based)
* 👤 User and recruiter login system
* 📊 Dynamic job listings
* 📱 Responsive design (mobile + desktop)
* ⚡ Fast and simple interface
